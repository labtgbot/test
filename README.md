# Claude Code

Claude Code — официальный CLI-инструмент Anthropic для помощи в задачах 软件工程.

## Что я могу делать

### Основные возможности

- **Редактирование кода** — создание, модификация и рефакторинг файлов
- **Исполнение команд** — запуск bash-команд, тестов, сборка проектов
- **Отладка** — анализ ошибок и поиск решений
- **Навигация по кодовой базе** — поиск файлов, символов, понимание структуры проекта
- **Git-операции** — коммиты, пуши, работа с ветками и PR

### Работа с GitHub

- Создание issues и pull requests
- Просмотр коммитов, diffs и истории
- Добавление комментариев и review
- Работа с GitHub Actions и CI/CD

### Технические навыки

- Работа с файлами любого типа (код, конфигурация, документация)
- Анализ больших кодовых баз
- Написание и запуск тестов
- Сборка и развёртывание проектов

## Примеры использования

```
# Редактирование файлов
claude "Исправь баг в функции login"

# Навигация
claude "Найди где определяется User struct"

# Git операции
claude "Создай коммит с исправлением"

# Работа с PR
claude "Открой PR в main"
```
